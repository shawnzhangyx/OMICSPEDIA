What is tag? How to use tag?

Tag types

Tags can be identified into three categories:

  1. Regular tag basic type of tags
  2. Software tag specialty tag that represents a software; used to identify posts that address the software.
  3. Workflow tags specialty tag that represents an OMICS workflow; used to identify posts that relate to a workflow. Workflow tags have hierarchical structures, and will show in the OMICS Portal.

Tag privileges

Verified users (:subject to change) can submit and edit new tags. Once a tag reaches a certain post count, it will be examined and become protected tags. protected tags is static and can no longer be edited.
