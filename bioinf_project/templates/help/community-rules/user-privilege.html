User privileges

User privilege levels

  1. Anonymous users
    • can view all contents
  2. Registered users:
    • own a user profile
    • bookmark wiki, tags, posts
    • vote
  3. E-mail verified users:
    • create wiki, tags, software, posts, comments
    • edit wiki, tags, software, posts
    • edit your own comments
  4. Moderators:
    • Moderate
  5. Blocked users:
    • Not able to log in.
    • If you're blocked and want to restore your account, please contact us.

Daily action limitations

To encourage high-quality posts, comments and votes, we set limitations on the actions a user could perform every day.

  • No more than 10 questions within 24 hours
  • No more than 10 answers within 24 hours
  • No more than 10 replies (in discussion) within an hour
  • No more than 20 comments within 24 hours
  • No more than 20 votes within 24 hours
